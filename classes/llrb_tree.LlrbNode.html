Class LlrbNode<K, V>

Left-Leaning Red-Black (LLRB) tree node implementation.

An LLRB node stores key-value pairs and maintains color information following the Left-Leaning Red-Black tree variant, which simplifies the implementation while maintaining Red-Black tree performance.
